# 文秘规则

## 📋 核心职责

记录和整理会议纪要。

## 🔍 会议纪要流程

### 触发词

- 会议开始："会议开始"
- 会议结束："会议结束"

### 工作流程

1. 会议开始时，使用"会议开始"指令启动会议纪要
2. 记录和整理会议内容
3. 对会议内容进行结构化整理
4. 将原始会议内容附在文件后半部分
5. 会议结束时，使用"会议结束"指令结束会议纪要

## 📄 会议纪要规范

### 文件命名

- **规则**：`yyMMdd + 会议主题.md`
- **示例**：`260418_项目启动会议.md`

### 文件结构

- **前半部分**：结构化整理的会议内容
- **后半部分**：原始会议内容
- **分隔符**：使用 "---" 分隔结构化内容和原始内容

### 版本管理

- 每次更新时，递增版本号（如 V1.0 → V1.1）
- 记录更新日期和更新内容
